# Android Gradle Plugin 9 Migration

Best practices and migration guide for Android Gradle Plugin (AGP) 9.x.

## Overview

AGP 9 introduces the Declarative Gradle DSL (Kotlin DSL only), drops legacy Groovy patterns, requires JDK 21, and changes several APIs. This guide covers the critical migration steps.

## Requirements

- JDK 21 (required)
- Gradle 8.10+ (required)
- Kotlin 2.0+ (strongly recommended)

## Key Breaking Changes

### 1. Namespace Required

The `namespace` property is now mandatory in every module:

```kotlin
// build.gradle.kts (app)
android {
    namespace = "com.example.myapp"  // REQUIRED – no longer inferred from manifest
    // ...
}
```

### 2. BuildConfig No Longer Generated by Default

```kotlin
android {
    buildFeatures {
        buildConfig = true  // opt-in if you use BuildConfig.DEBUG etc.
    }
}
```

### 3. Removed: compileSdkVersion / minSdkVersion (use compileSdk / minSdk)

```kotlin
// AGP 8 (old)
android {
    compileSdkVersion(34)
    defaultConfig {
        minSdkVersion(21)
    }
}

// AGP 9 (new)
android {
    compileSdk = 35
    defaultConfig {
        minSdk = 21
    }
}
```

### 4. Variant API Changes

```kotlin
// Old: afterEvaluate patterns are deprecated
// New: use androidComponents extension
androidComponents {
    onVariants(selector().withBuildType("release")) { variant ->
        variant.buildConfigFields.put(
            "SERVER_URL",
            BuildConfigField("String", "\"https://api.example.com\"", null)
        )
    }
}
```

## Migration Steps

1. Update `gradle-wrapper.properties` to Gradle 8.10+
2. Update AGP in `libs.versions.toml`: `agp = "9.0.0"`
3. Set JDK 21 in Android Studio: `File > Project Structure > SDK Location > Gradle JDK`
4. Add `namespace` to every module's `build.gradle.kts`
5. Replace `compileSdkVersion`/`minSdkVersion` with `compileSdk`/`minSdk`
6. Add `buildFeatures { buildConfig = true }` if needed
7. Run `./gradlew build --warning-mode all` and fix remaining warnings

## Version Catalog (libs.versions.toml)

```toml
[versions]
agp = "9.0.0"
kotlin = "2.0.21"
compileSdk = "35"
minSdk = "21"
targetSdk = "35"

[plugins]
android-application = { id = "com.android.application", version.ref = "agp" }
android-library = { id = "com.android.library", version.ref = "agp" }
kotlin-android = { id = "org.jetbrains.kotlin.android", version.ref = "kotlin" }
```

## Forbidden Patterns

- Do NOT use Groovy DSL (`.gradle` files) – migrate to Kotlin DSL (`.gradle.kts`)
- Do NOT use `compileSdkVersion` – use `compileSdk`
- Do NOT omit `namespace`
