# AGENTS.md – KoteGuard Agent Guidance

> This file is automatically injected into every KoteGuard worktree.
> It provides guidance for AI coding agents operating in this sandbox.

## You Are In a Safe Sandbox

This is an **isolated git worktree**. You are free to experiment, make mistakes,
and iterate. Your changes will be reviewed by a human before anything is merged.

## What You Should Do

1. Read `PLAN.md` carefully before making any changes.
2. Read `WORKSPACE.md` to understand the project structure and conventions.
3. Follow the tasks listed in `PLAN.md` in order.
4. Commit changes as you complete each task.
5. Write clear, descriptive commit messages.

## What You Must NOT Do

- Push to any remote branch.
- Create pull requests.
- Modify `.github/copilot-instructions.md` or `.github/instructions/security.instructions.md`.
- Access credentials or secrets (stubs are provided for reference only).
- Delete or move the `PLAN.md` or `WORKSPACE.md` files.

## Sensitive Files

Stub files exist for sensitive assets. These are **not real files** – do not
attempt to use them for signing or authentication.

## Getting Help

If you are uncertain about a task, leave a comment in the relevant file
explaining what you need clarification on.
